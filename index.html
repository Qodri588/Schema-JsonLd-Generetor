O que é

Schema JsonLD é a nova forma de informar as informações do site para a indexação dos buscadores.

Como usar

Copie o código gerado e cole dentro da tag head do seu site. 

<script type="application/ld+json">
{
	"@context": "http://www.schema.org",
	"@type": "product",
	"brand": "Marca",
	"name": "Nome",
	"image": "Imagem.jpg",
	"description": "Descrição",
	"aggregateRating": {
		"ratingValue": "Avaliação",
		"@type": "aggregateRating",
		"reviewCount": "Views"
	}
}
</script>

Para testar o que está sendo indexado do seu site, use o site www.googletools.com.br
